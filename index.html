Step 1 — paste your folder path
⚠ This looks like a folder name, not a full path. The path must start with a drive letter, e.g. C:\Users\YourName\Documents
  1. Open your folder in File Explorer
  2. Click the address bar at the top of the window — it will show the full path starting with C:\
  3. Press Ctrl+A to select it, then Ctrl+C to copy
  4. Paste it into the field above
Or drag a folder from File Explorer directly onto this window
